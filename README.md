# 📊 IT Tickets Analysis Dashboard

An interactive Excel dashboard analyzing **97,000+ IT service tickets** 
across multiple years — built entirely using Microsoft Excel.

> This project was created while learning Excel. No SQL or Power BI — 
> just Excel pushed to its full potential.

## 🔍 Project Highlights
- Analyzed 97,000+ IT service tickets across multiple years
- Built interactive dashboard using Pivot Tables, Pivot Charts & Slicers
- Designed executive KPIs: Total Tickets, Avg. Daily Tickets, 
  Resolution Time & Satisfaction Rate
- Deep analysis of trends, priorities, severity, issue types 
  & service performance

## 🛠️ Tools Used
`Microsoft Excel` `Pivot Tables` `Pivot Charts` `Slicers` 
`Advanced Formulas` `Data Cleaning` `Dashboard Design`

## 💡 Key Takeaway
Real understanding comes from building, breaking, and rebuilding.
This project proved that Excel alone — when used deeply — 
is a powerful analytics tool.
